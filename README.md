# vx-dga-l-meshlab

Paquete Debian para Vitalinux que proporciona un acceso directo a MeshLab AppImage.

## Descripción

Este paquete instala un lanzador para abrir MeshLab desde el escritorio Vitalinux. MeshLab es una herramienta de procesamiento y edición de mallas 3D orientada a análisis, limpieza y conversión de modelos.

## Requisitos

- Sistema Vitalinux compatible con paquetes Debian.
- Arquitectura `amd64`.
